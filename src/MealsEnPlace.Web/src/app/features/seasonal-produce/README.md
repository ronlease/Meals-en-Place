# Seasonal Produce

Displays what produce is in season for USDA Zone 7a with a view toggle.

## Backlog

- MEP-010 Seasonal Produce Guidance

## Route

`/seasonal-produce`

## Components

- **SeasonalProducePageComponent** — Table of produce with peak season date range. Button toggle switches between "In Season Now" and "Full Calendar" views. Zone badge displays "USDA Zone 7a".

## Services Used

- `SeasonalProduceService` — Get in-season and all seasonality windows
